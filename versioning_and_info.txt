Simulation script can be run via a command like:
$ESPRESSOMDPATH/build/pypresso simulation_script.py -no_obj 6120 -concentration 0.6 -no_per 25 -no_crowders 4080 -vdW 5 -vdW_ligand 10 -part_per_filament 15 -part_per_ligand 3 -path_data $PATH_TO_DATA_SAVE_LOCATION -MODE NEW -bonding_mode ftf

The authors recommend using 16 mpi ranks.

Running the provided simulation script, pressomancy saves data with all the contextual information in the .h5 format that is compatible with the analysis kernels in analysis toolbox. See pressomancy documentation for more details, samples for help. An example script how to launch any of the calculation kernels that produce the data for plots is provided in example_to_lauch_analysis.ipynb. This is a convenience to launch analysis in parallel.
